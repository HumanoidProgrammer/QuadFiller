# QuadFiller

A game inspired by Encompass/JezzBall, written in Python with Pygame.

## Install

```bash
pip install pygame
```

## Run

```bash
python quadfiller.py
```

## Controls

| Key | Action |
|-----|-------|
| Left-click | Draw a line |
| Right-click / Space | Toggle horizontal ↔ vertical |
| R | Restart |
| Esc | Quit |
